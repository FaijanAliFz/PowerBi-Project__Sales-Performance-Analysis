## 📊 Project Overview
This project presents an interactive Power BI dashboard analyzing company sales data at the transaction level. It provides a comprehensive view of sales performance, product contribution, and regional distribution.

---

## 🎯 Objectives
- Monitor overall sales performance  
- Analyze product category contribution  
- Identify top-performing customers  
- Evaluate regional sales distribution  
- Support data-driven business decisions  

---

## ⚙️ Dashboard Features
- KPI cards for total sales, orders, and quantity sold  
- Product category performance analysis  
- Customer contribution insights  
- Country-wise sales distribution  
- Interactive filters for dynamic analysis  

---

## 🛠️ Tools & Technologies Used
- Power BI  
- Data Modeling  
- DAX (Data Analysis Expressions)  
- Data Cleaning & Transformation  

---

## 📈 Key Insights
- Total sales reached **$10.03M** across **307 orders**  
- Approximately **99K units sold**  
- **Classic Cars** category generated the highest revenue  
- **United States** contributed the largest share of sales  
- Sales distribution varies significantly across regions  

---

## 🚀 Project Outcome
Developed an interactive dashboard that transforms raw sales data into **actionable insights**, enabling better tracking of performance, product strategy, and regional sales trends.
